# File Format Flask

A sample Flask API that shows how to read a variety of file formats in Python.

## Environment Setup

```bash
cp sampledotenv.env .env  # Set up env variables
pip install -r requirements.txt  # Install required packages
```

## How To Run

```bash
flask run
```
